# Boundaries

## What This Is

DeepGem-Config is a set of Claude Code skills for DeepGem Interactive's internal team. It helps PMs execute the change-centric planning methodology by automating document generation.

## What This Is NOT

**Not a general PM tool**
- Built specifically for DeepGem's methodology and templates
- Not designed to be shared or sold to other agencies
- References internal processes and pricing

**Not a replacement for judgment**
- AI generates drafts, humans review and approve
- Commercial terms always require human confirmation
- Client communication tone is guided but PM-reviewed

**Not integrated (yet)**
- Does not currently write to ClickUp
- Does not currently push to PandaDoc
- Output is markdown that gets manually transferred

## Out of Scope (for now)

These are future possibilities, not current capabilities:

- ClickUp API integration (reading/writing tasks)
- PandaDoc API integration (generating documents directly)
- Automated change order detection
- Client-facing features
- Multi-tenant or team management

## The Line

**AI can:**
- Generate complete document drafts
- Ask clarifying questions
- Reference templates and examples
- Calculate pricing and retainers
- Structure information from messy inputs

**AI must not:**
- Guess at pricing or rates
- Send documents to clients
- Make commitments on behalf of DeepGem
- Skip the review step
